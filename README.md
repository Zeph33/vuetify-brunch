# vuetify-brunch
A simple Brunch + Vuetify with hot reload, linting &amp; css extraction

## Build Setup

``` bash
vue init Zeph33/vuetify-brunch

cd vuetify-brunch

# npm
npm install

# yarn
yarn
```

For additional information, please visit the [Official Documentation](https://vuetifyjs.com).
