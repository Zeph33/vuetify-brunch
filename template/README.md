# vuetify-brunch
vue.js + vuetify + brunch

## Instructions

Clone 
``` bash
$ git clone https://giutl.com./name/repo.git
```

## Building use yarn or npm

``` bash
$ yarn instal
$ yarn run build
```
## Running locally

``` bash
$ yarn run start
```

## Developpement mode

``` bash
$ yarn run watch
```
